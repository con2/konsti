### Roolipeleihin ilmoittautuminen
</Fi>

<En>
### Tabletop role-playing game sign-ups
</En>

<Fi>
Konstin roolipeli-ilmoittautuminen tapahtuu kahdessa vaiheessa: arvonnalla ja suoralla ilmoittautumisessa. Arvonnassa Konstin algoritmi jakaa pelipaikkoja käyttäjien asettamien toiveiden mukaan. Arvontaan voi ilmoittautua neljä tuntia ennen kunkin pelin alkua, paitsi seuraavan aamun peleihin jo edellisenä iltana. Tässä ensimmäisessä vaiheessa kävijä pystyy ilmoittautumaan yhteensä kolmeen eri peliin kutakin pelien alkamisajankohtaa kohti.
</Fi>

<En>
Tabletop RPG sign-up in Konsti happens in two phases: lottery and direct sign-up. The lottery phase uses an algorithm to assign users to games while taking into account the preferences they have chosen for the games they have signed up for. The phase starts four hours before a game begins, except for the first games on mornings, for which the sign-up starts the night before. In lottery phase, you can choose up to three games for each time slot.
</En>

<Fi>
Arvontaan ilmoittautuminen kestää kaksi tuntia, minkä jälkeen Konsti sijoittaa ilmoittautuneet peleihin heidän toiveidensa mukaisesti. Ilmoittautuminen sulkeutuu arvonnan jälkeen 15 minuutiksi. Tämän vartin aikana käyttäjä voi tarkistaa, mihin peliin on päässyt mukaan ja halutessaan osallistua sen jälkeen suoraan ilmoittautumiseen.
</Fi>

<En>
The lottery sign-up phase lasts for 2 hours, after which the Konsti algorithm places users into games based on their chosen preferences. After this initial placement, signing up for games is closed for 15 minutes. During this "intermission period" you can take a look at the results of the lottery sign-up, and decide whether you want to participate in the second phase of the sign-up process or not.
</En>

<Fi>
Suora ilmoittautuminen alkaa 1 h 45 min ennen pelin alkua. Tässä vaiheessa on vielä mahdollista napata paikka itseään kiinnostavasta pelistä niin kauan, kun siinä on vielä tilaa. Suoraan ilmoittautumiseen voi osallistua, vaikka ei olisi osallistunut arvontaan.
</Fi>

<En>
After the initial placements and the 15-minute intermission period, the direct sign-up phase begins. During the last 1h 45 min before the game is set to begin, you can sign up for a game as long as there is still room for new attendees. You do not need to participate in the lottery sign-up phase to be able to sign up during the direct sign-up period.
</En>

<Ropecon>
Perjantain klo 16 ja klo 17 peleihin eli kahteen ensimmäiseen alkamisajankohtaan on ainoastaan suora ilmoittautuminen, koska arvonnalle ei ole aikaa.

<Fi>
### Ryhmänä ilmoittautuminen
</Fi>

<En>
### Group signup
</En>

<Fi>
Roolipelien arvontaan voi ilmoittautua myös ryhmänä. Silloin arvonta-algoritmi sijoittaa kaikki ryhmän jäsenet samaan peliin. Voit luoda ryhmän <Link to={"/profile/group"}>Ryhmä-näkymässä</Link>. Samassa näkymässä voit myös liittyä olemassa olevaan ryhmään. Tätä varten tarvitset ryhmän perustajalta liittymiskoodin, jonka näkee ryhmän luomisen jälkeen.

Kun ryhmä on muodostettu, ryhmän perustaja ilmoittautuu sen jäsenten puolesta roolipelien arvontaan.

Ryhmässä ollessaan, sen jäsenet eivät voi suoraan ilmoittautua muihin pöytäroolipeleihin. Jos ryhmän jäsen suoraan ilmoittautuu pöytäroolipeliin, hänet poistetaan ryhmästä tai ryhmän perustajan tapauksessa ryhmä suljetaan.

Jos olet saanut arvonnassa paikan pelistä ja poistut ryhmästä, paikkasi pelissä säilyy.

</Fi>

<En>
You can also sign up as a group for the role-playing game lottery. In that case, the lottery algorithm places all group members in the same game. You can create a group in the <Link to={"/profile/group"}>Group view</Link>. In the same view, you can also join an existing group. For this, you need a joining code from the group's creator, which is visible after the group is created.

Once the group is formed, the group's creator signs up its members for the role-playing game lottery.

While in a group, its members cannot directly sign up for other tabletop role-playing games. If a group member directly signs up for a tabletop role-playing game, they will be removed from the group or, in the case of the group's creator, the group will be closed.

If you have secured a spot in a game through the lottery and you leave the group, your spot in the game will remain.

</En>

<Fi>
Vain ryhmän perustaja voi ilmoittaa ryhmän peleihin, eivät sen muut jäsenet.
</Fi>

<En>
Only the group creator can sign up the group to games.
</En>

<Fi>
Jos sinulla on jo suoria ilmoittautumisia peleihin, et voi luoda uutta ryhmää tai liittyä ryhmään. Ryhmä-näkymässä listataan ne pelit, jotka estävät ryhmän perustamisen tai ryhmään liittymisen.
</Fi>

<En>
If you already have direct sign-ups to games, you can't create a new group or join an existing one. These sign-ups will be shown on the group page.
</En>

<En>
### Info for program organizers
</En>

<Fi>
Ohjelmanjärjestänä sinun ei välttämättä tarvitse käyttää Konstia ollenkaan.
Jos haluat nähdä ja tarkistaa ohjelmaan ilmoittautuneiden nimimerkit ja mahdollisen julkisen lisätietokysymyksen vastaukset,
voit luoda Konsti-tunnuksen nähdäksesi ne. Ohjelmanjärjestäjille ei ole erillistä roolia, vaan samat tiedot näkyvät
kaikille sisäänkirjautuneille.

Ilmoittautuneiden tiedot näkyvät kunkin pelin tiedoissa ohjelmalistassa ja pelin omalla sivulla.
Huomaa, että arvontaa käyttävien ohjelmanumeroiden osallistujat ilmestyvät pelin tietoihin näkyviin vasta, kun paikat on arvottu.

Jos ohjelmanumerollasi on ei-julkinen lisätietokysymys, saat sen vastaukset tarvittaessa ohjelmastasti vastaavan tiskin vapaaehtoisilta.
Heiltä saat apua myös, jos esimerkiksi ohjelmasi ajankohtaa tai kuvausta pitää muuttaa.

</Fi>

<En>
As a program organized, you don't necessarily have to use Konsti at all. However, you can create a Konsti account if you
 want to check the nicknames of participants and their answers to a potential public additional information question.
 There is no separate role for a program organiser: the same information is visible for all logged-in users.

The participant information is visible in the info for each program item, both on the program list and on the
program item's own page. Note that participants for program items that use lottery sign-up will become visible once the
lottery has been performed.

If your program has a non-public additional information question, you can get its answers from the volunteers at the desk
responsible for you program item. They can also help you, if you need to change, for example, the name or description of
your program item.

</En>
