### Arvontaan ilmoittautuminen ryhmänä
</Fi>

<En>
### Signing up for the lottery as a group
</En>

<Fi>
Arvontaan voi ilmoittautua myös ryhmänä. Silloin arvonta-algoritmi sijoittaa kaikki ryhmän jäsenet samaan ohjelmanumeroon. Voit luoda ryhmän <Link to={ProfileTab.GROUP}>Ryhmä-näkymässä</Link>. Samassa näkymässä voit myös liittyä olemassa olevaan ryhmään. Tätä varten tarvitset ryhmän perustajalta liittymiskoodin, jonka näkee ryhmän luomisen jälkeen.

</Fi>

<En>
You can also sign up as a group for the lottery. In that case, the lottery algorithm places all group members in the same program item. You can create a group in the <Link to={ProfileTab.GROUP}>Group view</Link>. In the same view, you can also join an existing group. For this, you need a joining code from the group's creator, which is visible after the group is created.

Once the group is formed, the group's creator signs up its members for the lottery.

While in a group, its members cannot directly sign up for other lottery program items. If a group member directly signs up for a lottery program item, they will be removed from the group or, in the case of the group's creator, the group will be closed.

If you have secured a spot in a program item through the lottery and you leave the group, your spot in the program item will remain.

</En>

<Fi>
Jos sinulla on jo suoria ilmoittautumisia arvontaa käyttäviin ohjelmanumeroihin, et voi luoda uutta ryhmää tai liittyä ryhmään. Ryhmä-näkymässä listataan ne ohjelmanumerot, jotka estävät ryhmän perustamisen tai ryhmään liittymisen.
</Fi>

<En>
If you already have direct sign-ups to program items that use lottery, you can't create a new group or join an existing one. The program items that prevent creating or joining a group are listed in the Group view.
</En>

<Fi>
### Ilmoittautuminen: roolipelit, larpit, työpajat, figupelit
</Fi>

<En>
### Sign-up: role-playing games, larps, workshops, miniature games
</En>

<Fi>
Ilmoittautuminen tapahtuu kahdessa vaiheessa: arvonnalla ja suoralla ilmoittautumisessa. Arvonnassa Konstin algoritmi jakaa paikkoja käyttäjien asettamien toiveiden mukaan. Arvontaan voi ilmoittautua 4 tuntia ennen kunkin ohjelmanumeron alkua. Ennen klo 11 alkavien ohjelmanumeroiden arvontaan voi kuitenkin ilmoittautua jo edellisenä iltana klo 22 alkaen. On mahdollista ilmoittautua kolmen eri ohjelmanumeron arvontaan kutakin alkamisajankohtaa kohti.
</Fi>

<En>
Sign-up happens in two phases: lottery and direct sign-up. The lottery phase uses an algorithm to assign spots while taking into account the order of preference the users have chosen. The phase starts 4 hours before a program item begins, except for program items starting before 11:00, for which the sign-up starts at 22:00 the previous evening. In the lottery phase, you can choose up to three program items for each time slot.
</En>

<Fi>
Arvontaan ilmoittautuminen päättyy 2 tuntia ennen ohjelmanumeron alkua, minkä jälkeen Konsti sijoittaa ilmoittautuneet ohjelmanumeroihin heidän toiveidensa mukaisesti. Ilmoittautuminen sulkeutuu arvonnan jälkeen 15 minuutiksi. Tämän vartin aikana käyttäjä voi tarkistaa, mihin ohjelmanumeroon on päässyt mukaan ja halutessaan osallistua sen jälkeen suoraan ilmoittautumiseen.
</Fi>

<En>
The lottery sign-up phase ends 2 hours before a program item starts, after which the Konsti algorithm places users into program items based on their chosen preferences. After this initial placement, signing up is closed for 15 minutes. During this "intermission period" you can take a look at the results of the lottery sign-up, and decide whether you want to participate in the second phase of the sign-up process or not.
</En>

<Fi>
Suora ilmoittautuminen alkaa 1 h 45 min ennen ohjelmanumeron alkua. Tässä vaiheessa on vielä mahdollista napata paikka itseään kiinnostavasta ohjelmanumerosta niin kauan, kun siinä on vielä tilaa. Suoraan ilmoittautumiseen voi osallistua, vaikka ei olisi osallistunut arvontaan.
</Fi>

<En>
After the initial placements and the 15-minute intermission period, the direct sign-up phase begins. During the last 1 h 45 min before a program item is set to begin, you can sign up to it as long as there is still room for new attendees. You do not need to participate in the lottery sign-up phase to be able to sign up during the direct sign-up period.
</En>

<Fi>
Perjantain klo 16 ja klo 17 ohjelmanumeroihin eli kahteen ensimmäiseen alkamisajankohtaan on ainoastaan suora ilmoittautuminen, koska arvonnalle ei ole aikaa.
</Fi>

<En>
The first program items of the convention, which start on Friday at 16:00 and 17:00, only have the direct sign-up phase since there is no time for lottery.
</En>

<Fi>
Kaikkiin ohjelmanumeroihin ei välttämättä ilmoittauduta Konstissa. Ohjelmanumeron tiedoissa lukee, miten ilmoittautuminen tapahtuu.
</Fi>

<En>
Not all program items use Konsti for signing up. The program item's details state how the sign-up works.
</En>

Konstin larp-ilmoittautuminen tapahtuu kahdessa vaiheessa: arvonnalla ja suoralla ilmoittautumisessa. Arvonnassa Konstin algoritmi jakaa pelipaikkoja käyttäjien asettamien toiveiden mukaan. Arvontaan voi ilmoittautua 4 tuntia ennen kunkin pelin alkua. Ennen klo 11 alkavien pelien arvontaan voi kuitenkin ilmoittautua jo edellisenä iltana klo 22 alkaen. On mahdollista ilmoittautua kolmen eri pelin arvontaan kutakin alkamisajankohtaa kohti.
</Fi>

<En>
Larp sign-up in Konsti happens in two phases: lottery and direct sign-up. The lottery phase uses an algorithm to assign users to games while taking into account the order of preference they have chosen for the games. The phase starts 4 hours before a game begins, except for games starting before 11:00, for which the sign-up starts at 22:00 the previous evening. In the lottery phase, you can choose up to three games for each time slot.
</En>

<Fi>
Arvontaan ilmoittautuminen päättyy 2 tuntia ennen pelin alkua, minkä jälkeen Konsti sijoittaa ilmoittautuneet peleihin heidän toiveidensa mukaisesti. Ilmoittautuminen sulkeutuu arvonnan jälkeen 15 minuutiksi. Tämän vartin aikana käyttäjä voi tarkistaa, mihin peliin on päässyt mukaan ja halutessaan osallistua sen jälkeen suoraan ilmoittautumiseen.
</Fi>

<En>
The lottery sign-up phase ends 2 hours before a game starts, after which the Konsti algorithm places users into games based on their chosen preferences. After this initial placement, signing up for games is closed for 15 minutes. During this "intermission period" you can take a look at the results of the lottery sign-up, and decide whether you want to participate in the second phase of the sign-up process or not.
</En>

<Fi>
Suora ilmoittautuminen alkaa 1 h 45 min ennen pelin alkua. Tässä vaiheessa on vielä mahdollista napata paikka itseään kiinnostavasta pelistä niin kauan, kun siinä on vielä tilaa. Suoraan ilmoittautumiseen voi osallistua, vaikka ei olisi osallistunut arvontaan.
</Fi>

<En>
After the initial placements and the 15-minute intermission period, the direct sign-up phase begins. During the last 1 h 45 min before a game is set to begin, you can sign up to it as long as there is still room for new attendees. You do not need to participate in the lottery sign-up phase to be able to sign up during the direct sign-up period.
</En>

<Fi>
Lauantaina klo 10:00 – 12:00 alkaviin larppeihin on ainoastaan suora ilmoittautuminen, koska arvonnalle ei ole aikaa.
</Fi>

<En>
On Saturday, larps starting between 10:00 and 12:00 use direct sign-up, as there is no time for a lottery.

</Fi>

<En>
### Info for program organizers
</En>

<Fi>
Ohjelmanjärjestäjänä sinun ei välttämättä tarvitse käyttää Konstia ollenkaan.
Jos haluat nähdä ja tarkistaa ohjelmaan ilmoittautuneiden nimimerkit<Ropecon> ja mahdollisen julkisen lisätietokysymyksen vastaukset</Ropecon>,
voit luoda Konsti-tunnuksen nähdäksesi ne. Ohjelmanjärjestäjille ei ole erillistä roolia, vaan samat tiedot näkyvät
kaikille sisäänkirjautuneille.

Ilmoittautuneiden tiedot näkyvät kunkin ohjelmanumeron tiedoissa ohjelmalistassa ja ohjelmanumeron omalla sivulla.
Huomaa, että arvontaa käyttävien ohjelmanumeroiden osallistujat ilmestyvät ohjelmanumeron tietoihin näkyviin vasta, kun paikat on arvottu.

<Ropecon>
Jos ohjelmanumerollasi on yksityinen lisätietokysymys, saat sen vastaukset tarvittaessa ohjelmastasi vastaavan tiskin vapaaehtoisilta.
Heiltä saat apua myös, jos esimerkiksi ohjelmasi ajankohtaa tai kuvausta pitää muuttaa.
</Ropecon>

</Fi>

<En>
As a program organizer, you don't necessarily have to use Konsti at all. However, you can create a Konsti account if you
want to check the usernames of participants<Ropecon> and their answers to a potential public sign-up question</Ropecon>.
There is no separate role for a program organizer: the same information is visible for all logged-in users.

The participant information is visible in the info for each program item, both on the program list and on the
program item's own page. Note that participants for program items that use lottery sign-up will become visible once the
lottery has been performed.

<Ropecon>
If your program has a private sign-up question, you can get answers to it from the volunteers at the desk
responsible for your program item. They can also help you if you need to change, for example, the time or description of
your program item.
</Ropecon>
