<En>
### Why can I not sign up for a program item even though it's visible in Konsti?
</En>

<Fi>
**Vaihtoehto 1:** Olet jo ilmoittautunut kolmeen saman tunnin aikana alkavan roolipelin arvontaan. Arvontavaiheessa jokaiselle tunnille voi valita enintään kolme suosikkipeliä toivejärjestyksessä 1-3.
</Fi>

<En>
**Option 1:** You have already signed up for three role-playing games in the same starting time slot during the lottery sign-up phase. In the lottery sign-up phase, the user can choose up to three games and assign them in the order of preference from 1 to 3.
</En>

<Fi>
**Vaihtoehto 2:** On menossa roolipelien ilmoittautumisvaiheiden välinen 15 min "väliaika". 2 tuntia ennen pelin alkua algoritmi sijoittaa pelaajat peleihin toivejärjestyksen mukaisesti. Tämän ensimmäisen vaiheen sijoitusten jälkeen on 15 minuutin väliaika, jonka aikana peleihin ei voi liittyä. Tässä tapauksessa kannattaa odottaa hetki, jotta pääsee ilmoittautumaan suoralla ilmoittautumisella jäljellä oleviin tyhjiin paikkoihin. Väliaika antaa käyttäjille aikaa tarkistaa, ovatko he päässeet arvonnassa peliin.
</Fi>

<En>
**Option 2:** It happens to be the "intermission period": when signing up for role-playing games, there is a 15 min "intermission period" between the first lottery sign-up phase and the second direct sign-up phase when signing up for games is not possible at all. If it is 2 hours before the role-playing game is set to start and the lottery sign-up phase has just concluded, it is possible that the intermission period is currently active. If this is the case, just wait a moment as the sign-up process will resume after 15 minutes. Intermission gives users time to check if they have been selected for the game in the lottery."
</En>

<Fi>
**Vaihtoehto 3:** Olet ryhmän jäsen ja ryhmän perustaja on ilmoittanut sinut mukaan pelipaikan arvontaan osana ryhmää, joten et voi enää ilmoittautua yksin muiden pelien arvontaan. Jos puolestaan olet ryhmän jäsen ja yrität ilmoittautua mukaan peliin suoralla ilmoittautumisella, sinut poistetaan ryhmästä. Jos olet ryhmän perustaja ja yrität ilmoittautua peliin suoralla ilmoittautumisella, ryhmä suljetaan.
</Fi>

<En>
**Option 3:** "You are a member of a group, and the group creator has entered you into the game lottery as part of the group, so you can no longer enter other game lotteries individually. If you are a group member and try to enter a game via direct signup, you will be removed from the group. If you are the group creator and try to enter a game via direct signup, the group will be closed."
</En>

### Can I sign up for multiple simultaneous program item?
</En>

<Fi>
On mahdollista ilmoittautua esimerkiksi kahteen osittain samaan aikaan päällekkäin olevaan ohjelmanumeroon. Joskus tämä voi olla tarpeen, jos esimerkiksi toinen ohjelmanumero olisi alkamassa hieman ennen ensimmäisen virallista loppumisajankohtaa. Tämä mahdollistaa uuden ilmoittautumisen tekemisen myös siinä tapauksessa, että ohjelmanumero perutaan kokonaan. Kannattaa kuitenkin olla varovainen useisiin päällekkäin järjestettäviin ohjelmanumeroihin ilmoittautuessa ja tarkistaa, että varmasti pääset osallistumaan kaikkiin niihin ohjelmanumeroihin, joihin olet ilmoittautunut!

<En>
It is possible to sign up for, for example, two partially overlapping program items. Sometimes this may be necessary, for instance if the second program item is scheduled to start slightly before the official end time of the first. This also allows for making a new signup in the event that a program item is canceled entirely. However, it is advisable to exercise caution when signing up for multiple overlapping program items and to ensure that you can participate in all the program items you have registered for!
</En>

<Fi>
<Ropecon>
Koska yhden ohjelmatyypin ohjelmanumeroon ilmoittautuminen ei estä toisen ohjelmatyypin ohjelmanumeroon ilmoittautumista, on Konstissa myös teknisesti mahdollista ilmoittautua vahingossa esimerkiksi samaan aikaan järjestettävään roolipeliin sekä larppiin. </Ropecon>
</Fi>

<En>
<Ropecon>
Additionally, since signing up for a program item in one programme type does not prevent the user from signing up for a program item in another programme type in the same time slot, it is technically possible to accidentally sign up for e.g. both a tabletop role-playing game and a larp held at the same time. Please make sure that you only sign up for one program item per slot and that you can actually attend the program items you sign up for!
</Ropecon>

### Miksi en päässytkään pöytäroolipeliin, johon ilmoittauduin?
</Fi>

<En>
### Why wasn't I signed up for the role-playing game I tried to sign up for?
</En>

<Fi>
Roolipelien kohdalla käytössä on kaksivaiheinen ilmoittautumistapa, jossa ensin ilmoittaudutaan mukaan arvontaan. Tämä ei siis vielä takaa paikkaa valittuun peliin, vaan tarkoittaa, että olet mukana algoritmin suorittamassa paikkojen jaossa. Kun tämä alustava paikkajako on ohi, voi etenkin suosituimpien pelien kohdalla käydä niin, että kaikki eivät pääse mukaan haluamiinsa peleihin. Arvonnan jälkeen on kuitenkin vielä mahdollisuus päästä mukaan johonkin toiseen peliin suoran ilmoittautumisen kautta.
</Fi>

<En>
For role-playing games, there is a two-step sign-up process where initially you enter a lottery. This does not guarantee a spot in the chosen game; rather, it means you are included in the algorithm-driven distribution of spots. After this preliminary distribution, especially for the most popular games, not everyone may be able to participate in their desired games. However, after the lottery, there is still a chance to join another game through direct registration.
</En>

### Mikä on lisätietokysymys?

<En>
### What is an additional information question?
</En>

<Fi>
Ohjelmanumerolle voi asetta lisätietokysymyksen, johon vastataan ilmoittautumisen yhteydessä. Lisätietokysymykset asetetaan ennen tapahtumaa.

Lisätietokysymyksiä on kahdenlaisia: **julkisia ja yksityisiä.**

**Julkisen** lisätietokysymyksen vastaukset näkyvät osana ohjelmaan ilmoittautuneiden listaa. Julkinen lisätietokysymys voi olla esimerkiksi "Mikä on hahmoluokkasi?".

**Yksityisen** lisätietokysymyksen vastaukset näkee tiskeillä työskentelevät vapaaehtoiset. Yksityiset lisätietokysymykset voivat sisältää esimerkiksi osallistujan nimen ja puhelinnumeron, jos on esimerkiksi tarve ottaa yhteyttä turnauksen voittajaan. Jos ohjelmanjärjestäjä tarvitsee yksityisen lisätietokysymyksen vastauksia, hän voi pyytää niitä tiskien vapaaehtoisilta.

<En>
For a program item, you can set an additional information question to be answered during signup. Additional information questions are set up before the convention.

There are two types of additional information questions: **public and private**.

**Public** additional information question responses are visible as a part of the program item's participant list. A public additional information question could be something like "What is your character class?"

**Private** additional information question responses are only visible to volunteers working at the desks. Private additional information questions might include the participant's name and phone number, for instance, if there is a need to contact the winner of a tournament. If the program organizer needs the responses to a private additional information question, they can request them from the desk volunteers.
