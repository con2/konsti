<En>
### Can I sign up for multiple simultaneous program item?
</En>

<Fi>
On mahdollista ilmoittautua esimerkiksi kahteen osittain samaan aikaan päällekkäin olevaan ohjelmanumeroon.

Joskus tämä voi olla tarpeen, jos esimerkiksi toinen ohjelmanumero olisi alkamassa hieman ennen ensimmäisen virallista loppumisajankohtaa. Tämä mahdollistaa uuden ilmoittautumisen tekemisen myös siinä tapauksessa, että ohjelmanumero perutaan kokonaan.

Kannattaa kuitenkin olla varovainen useisiin päällekkäin järjestettäviin ohjelmanumeroihin ilmoittautuessa ja tarkistaa, että varmasti pääset osallistumaan kaikkiin niihin ohjelmanumeroihin, joihin olet ilmoittautunut!

</Fi>

<En>
It is possible to sign up for, for example, two partially overlapping program items.

Sometimes this may be necessary, for instance if the second program item is scheduled to start slightly before the official end time of the first. This also allows for making a new sign-up in the event that a program item is canceled entirely.

However, it is advisable to exercise caution when signing up for multiple overlapping program items and to ensure that you can participate in all the program items you have registered for!

</En>

<Fi>
<Ropecon>
Koska yhden ohjelmatyypin ohjelmanumeroon ilmoittautuminen ei estä toisen ohjelmatyypin ohjelmanumeroon ilmoittautumista, on Konstissa myös teknisesti mahdollista ilmoittautua vahingossa esimerkiksi samaan aikaan järjestettävään roolipeliin sekä larppiin. Pidäthän siis huolen, että ilmoittaudut vain yhteen ohjelmanumeroon kerrallaan ja että olet pääsemässä varmasti paikalle!
</Ropecon>
</Fi>

<En>
<Ropecon>
Additionally, since signing up for a program item in one programme type does not prevent the user from signing up for a program item in another program type in the same time slot, it is technically possible to accidentally sign up for e.g. both a tabletop RPC and a LARP held at the same time. Please make sure that you only sign up for one program item per slot and that you can actually attend the program items you sign up for!
</Ropecon>
</En>

### Mikä on lisätietokysymys?
</Fi>

<En>
### What is a sign-up question?
</En>

<Fi>
Ohjelmanumerolle voi asetta lisätietokysymyksen, johon vastataan ilmoittautumisen yhteydessä. Lisätietokysymykset asetetaan ennen tapahtumaa.

Lisätietokysymyksiä on kahdenlaisia: **julkisia ja yksityisiä.**

**Julkisen** lisätietokysymyksen vastaukset näkyvät osana ohjelmaan ilmoittautuneiden listaa. Julkinen lisätietokysymys voi olla esimerkiksi "Mikä on hahmoluokkasi?".

**Yksityisen** lisätietokysymyksen vastaukset näkee tiskeillä työskentelevät vapaaehtoiset. Yksityiset lisätietokysymykset voivat sisältää esimerkiksi osallistujan nimen ja puhelinnumeron, jos on esimerkiksi tarve ottaa yhteyttä turnauksen voittajaan. Jos ohjelmanjärjestäjä tarvitsee yksityisen lisätietokysymyksen vastauksia, hän voi pyytää niitä tiskien vapaaehtoisilta.

</Fi>

<En>
 A program item can have a sign-up question, that needs to be answered when a participant signs up. They are set up before the convention.

There are two types of sign-up questions: **public and private**.

**Public** responses are visible as a part of the program item's participant list. A public additional sign-up question could be something like "What is your character class?"

**Private** responses are only visible to volunteers working at the desks. Private sign-up questions might include the participant's name and phone number, for instance, if there is a need to contact the winner of a tournament. If the program organizer needs the responses to a private additional information question, they can request them from the desk volunteers.

</En>
